# Security Policy

## Supported Versions

This is a documentation-repository with no runnable code. There are no versioned releases to patch.

## Reporting a Vulnerability

If you discover any security-related issues (such as malicious external links or phishing URLs in the documentation), please:

1. **Do not** open a public issue.
2. Report the problem via GitHub's [Private Vulnerability Reporting](https://github.com/Thinkright20/Profile-Badges/security/advisories/new).

## Scope

This repo contains only Markdown documentation and external image references. Security concerns are limited to malicious links or content inaccuracies.
